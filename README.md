# Weatherly

A simple React application where the user enters a location and gets the current weather for that place using the OpenWeather API.

## Features

* Search weather by city or location
* Shows temperature, humidity, wind, and basic conditions
* Clean and minimal interface

## Tech Used

* React
* Vite
* OpenWeather API

## How to Run

1. Clone the project:

   ```bash
   git clone https://github.com/your-username/weatherly.git
   ```
2. Install dependencies:

   ```bash
   npm install
   ```
3. Create a `.env` file:

   ```
   VITE_WEATHER_API_KEY=your_openweather_api_key
   ```
4. Start the app:

   ```bash
   npm run dev
   ```

Open the app in your browser at the URL shown in the terminal.
